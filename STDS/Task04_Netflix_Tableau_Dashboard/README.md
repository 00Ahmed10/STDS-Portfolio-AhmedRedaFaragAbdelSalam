---

## Dataset Description

The dataset contains metadata about Netflix titles, where each row represents **one movie or TV show** available on Netflix.

### Key Columns

* **show_id**: Unique identifier for each title
* **type**: Content type (Movie or TV Show)
* **title**: Name of the movie or TV show
* **director**: Director(s) of the content
* **cast**: Main actors
* **country**: Production country or countries
* **date_added**: Date when the title was added to Netflix
* **release_year**: Original release year
* **rating**: Age rating (e.g., TV-MA, TV-14, PG-13)
* **duration**: Movie runtime (minutes) or number of seasons
* **listed_in**: Genre(s)
* **description**: Short description of the content
